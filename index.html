Firefox Brand + Design Assets

Hello!

November 1st, 2019 – The Firefox UX and Brand teams are currently working on consolidating a number of disparate design assets. This work will likely continue into early 2020.

  • Photon Icons

    Icons created as part of the Photon Design System. These are still usable in browser, app and marketing properties.

  • Photon Design System

    The design system developed in conjunction with the Firefox Quantum release. This system has useful content guidelines and UI guidelines for the Firefox Browser.

  • Protocol Component Library

    A CSS framework and usage guide maintained by the WWW team for use on mozilla.org. This framework is currently being adapted to be more widely applicable for web applications.

  • Firefox User Research

    An up-to-date reference for user research reports at Mozilla.

  • Firefox UX Blog

    Articles about research and design activities at Mozilla.

  • Firefox Design Values

    The values of the Firefox UX team that inform our design and decision-making processes.

  • Careers

    Interested in joining a fast paced, multidisciplinary design team? Come say hi.
